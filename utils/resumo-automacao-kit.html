1. Diagnóstico e Problema Central

O objetivo é automatizar o acesso à plataforma Belenus para replicar fielmente os preços de kits personalizados, eliminando a discrepância atual de ~30% identificada em kits de pequeno porte.

Desvio Detectado:
  • Cálculo Atual: R$ 6.687,20
  • Preço Real (Belenus): R$ 8.704,76
  • Gap: R$ 2.017,56 (Necessita ajuste de motor)

2. Plano de Ajuste da Engine (model.js)

A transição do modelo de "Soma de Custos Unitários" para "Modelo de Base Calibrada por Fatores".

Antiga Lógica:

total = custoMod + custoInv + custoEst + custoProt + frete;

Nova Lógica Recomendada:

total = (baseKitCalibrada * fatorFaixaKwp * fatorLogistico * fatorComplexidade) * fatorPremium;

4. Implementação Técnica (Auto-Calibração)

A estrutura abaixo deve ser implementada no Cloudflare D1 para garantir que o sistema aprenda com novos PDFs carregados.

-- Tabela de Parâmetros Ativos
CREATE TABLE parametros_calibracao (
id INTEGER PRIMARY KEY,
faixa_kwp REAL,
fator_ajuste REAL DEFAULT 1.0,
last_update DATETIME
);

-- Lógica de Ajuste (Media Móvel)
fatorCalibracao = media (preco_real_pdf / preco_calculado_engine);
